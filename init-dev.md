

# 前端在线学习网站
制作一个前端学习网站，希望快速掌握前端编程知识和技巧，以应对即将面临的前端岗位求职面试。

### Tech Stack技术栈 (固定,不要自行更换)
- **Framework**: React 18 + Next.js 15 (App Router) + TypeScript (strict mode)
- **UI**: Tailwind CSS + shadcn/ui
- **State**: Mobx / Redux (client state) + TanStack Query (server state)
- **DB**: SQLite + Prisma (本地开发), 方便后续迁移 Postgres
- **Code editor**: Monaco Editor (`@monaco-editor/react`)
- **Code runner**: WebContainer 沙箱 (仅 JS/TS 题目,MVP 不跑 Node)
- **Markdown + 代码高亮**: `react-markdown` + `shiki`
- **Testing**: Vitest (单元) + Playwright (E2E, 仅关键路径)
- **Package manager**: pnpm

### 题目说明
包括单选题、多选题、填空题、编程题。
用户可以在线做题，系统会立即自动评分并提供反馈：有答案参考和解题思路的详细解析，还有相关题目推荐。题目的评分系统根据该题目的难度进行预设。
用户可以选择反馈是否已经学会、跳过、收藏、上一题、下一题。
- 题目类型：
    - 单选题：用户从多个选项中选择一个正确答案。
    - 多选题：用户从多个选项中选择一个或多个正确答案。
    - 填空题：用户在文本框中输入答案。
    - 编程题：用户在在线编辑器中编写代码，代码高亮，并可以在线执行。提交后系统会根据预设的测试用例进行自动评分。编程题需要有3个以上的test case。
    - 打印结果题：用户需要根据题目要求，输出特定的结果，系统会根据输出结果进行评分。
    - 问答题：题目给出一个问题，用户输入一长段文字回答，系统根据输入的内容与结果进行对比评分。

答案展示页需要展示是否正确，评分，以及用户答案与标准答案的对比展示，以及各个test case及结果。

### 题库查看
可以查看题库，可以按照多种方式排序：最近学习、最多学习、最多正确、最多错误、最差评分。
可以进行筛选：类型、错题等等。

## 学习计划
系统进行学习计划制定时候，严格按照艾宾浩斯记忆曲线的原理进行制定。
系统会自动提供1周学习计划及每日学习计划。
用户可以自己手动生成新的临时学习计划。

## 模拟面试
模拟面试模式，用户可以随机选择n个题目/固定时长进行模拟面试，系统会根据时间限制和评分标准进行评估和打分。
